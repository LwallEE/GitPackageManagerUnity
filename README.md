# GitPackageManagerUnity
# Features
- Manage git packages for Unity through UGUI.
- Support manage versions of package.
- Support manage dependencies of package.
- Support public and private repository.
- Support multiple packages in one repository.
# Install
- Install via git url in UPM: https://github.com/LwallEE/GitPackageManagerUnity.git?path=/Assets/GitPackageManager#main
